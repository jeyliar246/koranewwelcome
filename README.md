# Kora AI Welcome Page

A beautiful, modern welcome page for Kora AI with typewriter animation and glass blur effects.

## Features

- ✨ Beautiful glass morphism design
- ⌨️ Typewriter animation showcasing example prompts
- 🎨 Animated gradient background
- 📱 Fully responsive design
- 🚀 Built with Vite for fast development

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will open at `http://localhost:3000`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Technologies

- Vite
- Vanilla JavaScript
- CSS3 (Glass morphism, animations)
